the methodology

Blueprint — brownfield audit-driven delivery.

This site is the latest output of Blueprint — the brownfield audit-driven methodology I'm developing in wip/blueprint and applying to projects including Rally HQ, Signal Dispatch, and this one.

Blueprint defines a seven-stage pipeline; brownfield projects follow it differently than greenfield. v3 of this site is a brownfield Pattern B initiative — diagnose-prescribe-design-prototype the new state against the live product.

predecessors + influences

Where the patterns came from. Citing the lineage.

The site shape owes most to Brian Lovin, Ryan Lopopolo, and Geoffrey Litt — capability sentence plus named receipt list is their pattern (per blueprint/research/competitive/peer-cohort.md, Archetype B).

The methodology-as-meta-receipt move is borrowed from Will Larson's body of work — systematized practitioner methodology applied to one's own surface. Blueprint is, in shape if not in topic, "Larson's pattern applied to AI-native delivery."

The harness-engineering vocabulary on this site predates the OpenAI Feb 2026 piece but was named publicly first by Ryan Lopopolo. Citing the coiner here so the lineage stays straight; the contribution this site claims is the commerce-specific application, not the term.

The body-in-serif move (cohort-divergent) is calibrated against the writer-cohort — Julian Lehr's lifelog, the magazine-essay tradition. The cyanotype color is direct material reference to Sir John Herschel's 1842 architectural drafting reproduction process.

Anti-slop posture borrowed from letspepper.com + flickdaymedia.com — sites I've previously fought this same fight on. Big confident type, saturated color, not-muted-tasteful, type-driven. Form reinvented because architect-who-writes is a different genre than sport-cultural-brand.

what changed from v2

v3 closes the diagnoses, then differentiates.

Identity frame
v2 read hobbyist polymath (Photography + Music + Writing + Building as 4 equal pursuits). v3 is architect-primary; creative pursuits live in subdomains, surfaced as nav links only. Closes D1.
Receipt density
v2 surfaces 4 products. v3 surfaces 11 CURRENT receipts, topic-grouped. None of the audited cohort sites has this many shipped surfaces. Closes D8.
Contact path
v2 had an unstructured mailto and an unsurfaced Cal.com. v3 has a single general inquiry CTA. Closes D4.
/ai surfacing
v2's /ai subsection (5 pages, 1,377 lines) was invisible from /. v3 surfaces it from nav. Closes D2.
Design system
v2 lived in drift between DESIGN.md (violet content-forward) and live / (lime bento spectacle). v3 retires both; new system from clean slate. Closes D5.
