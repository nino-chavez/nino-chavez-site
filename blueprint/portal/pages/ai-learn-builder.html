Who this is for

This path is for you if

  • You're a developer who wants to ship AI-powered software
  • You learn by building, not by watching tutorials
  • You want production-ready code, not demo projects
  • You're comfortable with JavaScript/TypeScript and web development

Probably not if

  • You want a curriculum to passively consume
  • You're collecting credentials — the artifact here is a deployed app, not a certificate

The levels

0 Orientation week 1

Set up the working environment and ship a first AI-assisted change to a real repository — the goal is contact with the loop, not theory.

1 Ship Your First AI App weeks 2–3

A small end-to-end application, deployed. The worked example is real: the reference apps this track links are built on the same stack this level uses.

2 Specs, Databases & Workflows weeks 4–6

Spec-first building — this is where the installable workflow above stops being a link and becomes your daily loop.

3 Multiple Models & Tool Use weeks 7–9

Agentic patterns — reflection, tool use, planning, multi-agent — studied against a shipped implementation rather than toy examples.

4 Production Systems & Observability weeks 10–12

What survives contact with real users: observability, governance, and the error-recovery patterns the reference apps document in their own ADRs.
